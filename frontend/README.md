# GreenFArmiq

GreenFarmIQ frontend (Vite + React).

Key scripts:

- Install deps: `npm install`
- Dev server: `npm run dev`
- Build: `npm run build`
- Preview: `npm run preview`

Deployment (Netlify):

- Publish directory: `dist`
- SPA redirects: `/public/_redirects` includes `/* /index.html 200`
- Optional config: `netlify.toml`
